## Project Overview
The Weather Dashboard is a React application that fetches live weather data for a user-specified city using the OpenWeatherMap API. It displays current weather conditions, including temperature, humidity, and weather description. A simulated 5-hour temperature trend chart visualizes changes, demonstrating how raw data can be transformed into actionable insights. Optional predictive functionality simulates next-day temperature forecasting.

## Problem Solved
Provides a clear and interactive way to view weather data and trends. Demonstrates key concepts in data science workflows: data retrieval, transformation, visualization, and insight generation.

## Features
- Enter a city to fetch live weather data
- Display temperature, humidity, and condition
- Simulated 5-hour temperature trend chart
- Optional next-day forecast simulation
- Responsive and interactive design
